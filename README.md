## 代码提交
### 提交代码的正确性
在根目录生成`.pre-commit-config.yaml`，运行`pre-commit install`(需要安装`pip install pre-commit`)，以后`git commit`时就会自动做这一系列的检查，保证提交代码的最基本的正确性。
